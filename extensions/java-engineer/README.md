# java-engineer

Java 21 language features, idioms, and coding standards. Covers records, sealed classes, pattern matching, virtual threads, streams, Optional, generics, exception handling, and naming conventions. Pure Java — no framework-specific content.

## Features
- Deep knowledge of Java 21 features (Virtual Threads, Pattern Matching).
- Clean code patterns and idiomatic Java development.
- Performance and memory management best practices.

## Installation
Add this extension to Junie via your configuration settings.
