# kotlin-engineer

Complete Kotlin development guide: coroutines, Flow, DSL patterns, idiomatic Kotlin, Ktor server, Android Compose, Kotlin Multiplatform, architecture review, performance, and debugging. Self-contained — works with or without Spring Boot.

## Features
- Expert knowledge in Coroutines and Flow API.
- Kotlin Multiplatform (KMP) support.
- Idiomatic Kotlin development and performance tuning.

## Installation
Add this extension to Junie via your configuration settings.
