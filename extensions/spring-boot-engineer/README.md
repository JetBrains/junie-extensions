# spring-boot-engineer

Complete Spring Boot 3.x guide: REST APIs, Spring Data JPA, Spring Security 6, WebFlux reactive, Kotlin patterns, event-driven architecture, Resilience4j, Spring Cloud, and testing. Single skill covering the full stack.

## Features
- Spring Boot 3.x modernization.
- Reactive stack (WebFlux) and Security 6.
- Cloud native and event-driven patterns.

## Installation
Add this extension to Junie via your configuration settings.
